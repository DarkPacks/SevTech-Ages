These are the functions which can be called on an DimensionData object.



setDimensionType:
    Arguments:
        String
    Usage:
        dimension type
    Notes:
        Sets the dimension type. Vanilla options are "overworld", "the_nether", "the_end"

setDimensionID:
    Arguments:
        int
    Usage:
        dimension id
    Notes:
        Sets the dimension type. Vanilla options are "overworld", "the_nether", "the_end"
